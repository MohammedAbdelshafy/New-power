Content Intelligence — YouTube / Instagram / TikTok
How it works
Drop any YouTube or Instagram URL. The ops room will:

1. Extract full metadata (title, views, likes, tags, description)
2. Pull the transcript/captions if available
3. Detect: hooks, CTAs, structural techniques, pacing, engagement signals
4. Generate ranked implementation tasks for your projects
5. Save a patch you can apply to any session's CLAUDE.md
Run from terminal
# YouTube
python3 ops-room/intelligence/analyze-content.py https://youtube.com/watch?v=XXXX

# Instagram Reel
python3 ops-room/intelligence/analyze-content.py https://instagram.com/reel/XXXX

# Then extract ranked techniques:
python3 ops-room/intelligence/extract-techniques.py
Detected Technique Categories
Hooks: Curiosity, Pattern Interrupt, Transformation Story, Anti-advice, Revelation
CTAs: Subscribe/Follow, Link/Navigate, Lead-gen, Comment Bait
Structure: Pacing (words/min), Format (short/mid/long-form), Sequential, Storytelling, Data authority
Engagement: Like-to-view ratio, tag density, SEO signals
Operations Queue
Queue Directory Structure
ops-room/operations/
  queue/ ← incoming analyses (auto-saved by analyzer)
  active/ ← move here when you start implementing
  completed/ ← move here when done
Each operation is a JSON file with: URL, platform, metadata, analysis, implementation_tasks[], status.
Viral Operation Format
# To manually queue a viral AI prompt:
# Create: ops-room/operations/queue/YYYYMMDD-viral-op.md

# Title: [describe the technique]
# Source: [URL or description]
# Target: [which session/project to apply it to]
# Intent: [what problem does this solve]
# Implementation: [how to apply it]
